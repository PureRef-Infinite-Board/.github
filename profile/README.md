# PureRef Software — Portable Infinite Reference Board

## About PureRef Software

PureRef Software provides an infinite, boundless canvas specifically designed for collecting and organizing visual references. PureRef Software lets you drag and drop images from anywhere — web browsers, file managers, screenshots — and arrange them freely on an endless workspace. Zoom in and out, pan across the canvas, flip images horizontally and vertically, group by color, lock transparency, and auto‑arrange scattered images. PureRef Software saves your entire workspace — image positions, zoom level, window placement — and can stay on top of other applications while you work. PureRef Software enables artists, designers, and creatives to gather inspiration without worrying about canvas size or document limits. PureRef Software runs on portable storage, USB drives, and local user folders. PureRef Software is trusted by concept artists, illustrators, game developers, mood board creators, and visual researchers worldwide.
